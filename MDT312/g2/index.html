1. พี่ทรายต้องการเขียน website ที่แสดงรูปภาพ 2 รูป (ตามตัวอย่าง) แต่เมื่อพี่ทรายทำเสร็จแล้ว กลับไม่ได้ผลตามที่ต้องการ ให้นักศึกษาช่วยพี่ทรายหาที่ผิด และอธิบายว่าพี่ทรายทำผิดที่จุดใดบ้างใน JavaScript file (5 คะแนน ส่งเป็นคำอธิบายอย่างชัดเจนว่าผิดที่จุดใดนะคะ)

2. จากโค้ดที่ให้ไป ให้นักศึกษาสร้าง website ให้สมบูรณ์ตามตัวอย่าง นั้นคือ เมื่อเราเริ่มรัน website จะแสดงผลดังตัวอย่าง แต่ถ้าเรากดที่ปุ่ม Delete ข้อมูลที่อยู่ในแถวนั้นจะถูกลบออกไปทั้งบนหน้าเว็บและใน json file (5 คะแนน ห้ามแก้โค้ดที่ให้ไป เพิ่มไฟล์ใหม่ได้อย่างเดียว เดี๋ยวตรวจไม่ได้ และส่งเฉพาะไฟล์ที่เพิ่มนะ)
(Note: ใช้คำสั่ง delete data['row']; สำหรับลบข้อมูลตัวที่ row ออกจาก JavaScript Object)
